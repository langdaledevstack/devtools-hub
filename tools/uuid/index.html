UUID Version Guide

  • Version 4 (Random) - Best for most use cases, randomly generated
  • Version 1 (Time-based) - Includes timestamp and MAC address
  • Version 3 (MD5 Hash) - Namespace + name hashed with MD5
  • Version 5 (SHA-1 Hash) - Namespace + name hashed with SHA-1 (more secure than v3)
